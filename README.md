A tiny library for processing Erlang BEAM file in Rust.

Documentation
-------------

See [RustDoc Documentation](https://docs.rs/beam_file)

The documentation includes some examples.

Installation
------------

Add following lines to your `Cargo.toml`:

```toml
[dependencies]
beam_file = "0.2"
```
